# **SCT_WD_1**
Responsive Landing Page
A responsive, animated landing page built with pure HTML, CSS, and vanilla JavaScript. No frameworks. No dependencies.

**Features**

Scroll-aware navbar — transforms on scroll with blur backdrop and shadow
Animated hero section — staggered entrance animations, floating decorative elements, parallax background
Live stat counters — numbers animate into view using IntersectionObserver
Feature cards — fade-in on scroll with hover lift and gradient top border
About section — embedded photo with color overlay and zoom-on-hover
CTA section — radial gradient backgrounds with call-to-action button
Mobile menu — full-screen overlay with hamburger toggle
Scroll-to-top button — appears after scrolling 300px
Custom scrollbar — styled to match the brand palette


**Project Structure**

SCT_WD_1/
├── landing page.html   # Single self-contained file (HTML + CSS + JS)
└── README.md           # Project documentation

**Tech Stack**

HTML5
CSS3 — custom properties, keyframe animations, grid, flexbox
JavaScript — IntersectionObserver, scroll events, DOM manipulation
